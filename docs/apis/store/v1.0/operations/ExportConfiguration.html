Export Configuration

Export details related to an Application.
GET https://apis.wso2.com/api/am/store/v1.0/export/applications

This operation can be used to export details related to a perticular application.

OAuth 2.0 Scope

apim:subscribei

Request

GET https://127.0.0.1:9292/api/am/store/v1.0/export/applications?query=xxx Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8

Response

HTTP/1.1 200 OK Connection: keep-alive Content-Disposition: attachment; filename="exported-applications.zip" Content-Type: application/zip

Sample CURL

curl -k -H "Authorization: Bearer ae4eae22-3f65-387b-a171-d37eaa366fa8" https://localhost:9292/api/am/store/v1.0/export/applications?appId=xxx > exported-application.zip
